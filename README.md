# Cristin O'Connor, Front End Engineer

This site requires node version 14.

# Setup

1. `nvm use 14.0.0`

# Local Development

1. `npm install --legacy-peer-deps`
2. `npm run develop`
3. Go to _[http://localhost:8000](http://localhost:8000)_.

# Deployment

Netlify is used.
